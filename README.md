## 🚀 Google Colab で学ぶ

すべての notebook は Google Colab で実行できます。  
環境構築は不要で、ブラウザだけで学習できます。

### 🔧 Colab の使い方
1. 各章のノートブック（vol01～10のnotebook.ipynb）を開く  
2. ノートブック一番上の「Open in Colab」をクリック  
3. 上部メニュー → **ランタイム → ランタイムのタイプを変更**  
4. ハードウェアアクセラレータを **GPU** に設定  
5. 上から順にセルを実行するだけ

### 重要：ノートブックは “閲覧専用” です。
そのままでは編集できないので、必ず 自分の Google ドライブにコピーしてから 編集してください。

📄 自分用にコピーする手順
- Colab でノートブックを開く
- 画面左上の 「ファイル」→「ドライブにコピーを保存」 を選ぶ
- コピーされたノートブックを自由に編集できます
